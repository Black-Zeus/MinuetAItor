===============================================================================

string obligatorio → usar valor entregado por el sistema o ""

string obligatorio → generar timestamp actual en formato ISO 8601 UTC:

ejemplo: 2026-03-01T15:30:00Z

string obligatorio → usar el valor de generalInfo.preparedBy

string obligatorio → usar "1.0"
